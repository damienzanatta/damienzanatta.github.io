Projet 1

Configuration des paramètres initiaux d'un commutateur

TP Packet Tracer 2.5.5 – BTS SIO SISR

Dans le cadre de ma formation en BTS SIO option SISR, j’ai réalisé un TP Cisco Packet Tracer portant sur la configuration initiale d’un commutateur. L’objectif était de préparer un commutateur Cisco à son utilisation dans une infrastructure réseau en configurant les paramètres de base : nom de l’équipement, mots de passe, bannière d’avertissement, chiffrement et sauvegarde de la configuration.

Objectifs

  • Renommer le commutateur.
  • Sécuriser l’accès console.
  • Protéger l’accès au mode privilégié.
  • Chiffrer les mots de passe.
  • Ajouter une bannière MOTD.
  • Sauvegarder la configuration.

Environnement technique

Logiciel

Administration et sécurisation réseau

Topologie

Étapes réalisées

Étape 1 – Vérification de la configuration initiale

J’ai commencé par afficher la configuration active du commutateur avec la commande show running-config. Cette étape m’a permis de vérifier l’état initial du commutateur avant toute modification. 

show running-config

Étape 2 – Configuration du commutateur

J’ai ensuite configuré les principaux paramètres du commutateur : le nom d’hôte, l’accès console, l’accès au mode privilégié, le chiffrement des mots de passe et la bannière d’avertissement. 

hostname S1
line console 0
password [mot_de_passe]
login
enable secret [mot_de_passe_secret]
service password-encryption
banner motd "L'acces a ce systeme est reserve aux personnes autorisees !"

Étape 3 – Vérification de l’accès sécurisé

Après la configuration, j’ai vérifié que le mot de passe était bien demandé à la connexion et que la bannière MOTD s’affichait correctement. Cette vérification confirme que l’accès au commutateur est protégé.

Étape 4 – Vérification finale et sauvegarde

J’ai ensuite affiché la configuration finale afin de contrôler les paramètres appliqués, puis j’ai sauvegardé la configuration pour qu’elle soit conservée après redémarrage. 

show running-config
copy running-config startup-config

Résultat obtenu

À la fin du TP, le commutateur S1 dispose d’une configuration initiale sécurisée : nom d’hôte configuré, accès console protégé, mode privilégié sécurisé, mots de passe chiffrés, bannière MOTD affichée et configuration sauvegardée.

  • Nom d’hôte configuré.
  • Accès console protégé.
  • Mode privilégié sécurisé.
  • Mots de passe chiffrés.
  • Bannière MOTD affichée.
  • Configuration sauvegardée.
